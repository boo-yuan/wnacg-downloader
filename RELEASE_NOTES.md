# WNACG Downloader v1.0.0 正式版发布 🎉

历经无数次的底层推翻与重构，**WNACG Downloader** 终于迎来了它的首个工业级正式版：**v1.0.0**！
在这个版本中，我们彻底抛弃了“玩具级”的爬虫脚本思维，为您带来了一个真正的“下得爽、存得稳、绝不封号”的现代化桌面端下载引擎。

---

## 🌟 核心亮点 (Key Features)

### 1. 纯净单文件，小白“零配置”直装
告别繁琐的 Python 环境配置！本次发布我们启用了全新的自动化打包流水线。
您只需要下载 WNACG-Downloader.exe，把它放在任意喜欢的文件夹里，双击即可直接使用！连解压的步骤都为您省去了。

### 2. “不封号”的智能排队下载
曾经挂机一整晚，早上醒来发现 IP 被网站封锁了？
我们在底层重写了网络调度器，引入了类似人类操作的“随机抖动延迟”。这就相当于有无数个机器人在帮您排队拿图，每个机器人的动作都极其自然，彻底瓦解高级防爬系统（WAF），长时间挂机稳如泰山。

### 3. 断电重启接着下，半截图“零容忍”
- **断点续传**：所有的进度都会被数据库毫秒级地记录下来，哪怕电脑突然断电重启，重新打开软件后也会自动接着断点继续下载。
- **图像防损坏**：告别恶心的“半身图”。程序在保存完每一张漫画后，都会强制“看”一眼它的底部。如果发现没下载完（损坏），会毫不犹豫地扔掉重下，确保您的本地画廊 100% 完美无瑕。

### 4. 彻底解决国内“更新难”
每次出新版本，都要挂代理去 GitHub 龟速下载？
现在不需要了！我们在软件的“设置”里内置了国内公益极速镜像源（如 kkgithub）。以后只要点一下“检查更新”，软件会自动、满速地把新版本拉取回来并完成自我替换。

### 5. 极致优雅的现代化界面
不仅好用，还要好看。整个界面基于 Windows 11 的亚克力半透明风格打造，支持深色/浅色模式无缝切换。更支持 **鼠标框选、Shift 连选、双击卡片暂停/恢复**，所有的操作都顺滑得像在使用系统原生的应用。

---

## 🛠️ 安装指南 (How to Install)

1. 点击下方的 **Assets** 列表。
2. 下载唯一的 WNACG-Downloader.exe 文件。
3. 把它丢进一个专属于您的文件夹里（程序运行时会自动在同级目录生成 downloads 存放漫画）。
4. 双击运行，开启您的自动化收图之旅！
