## 🎉 简介 (Introduction)

WNACG Downloader 是一个为 WNACG 漫画爱好者量身定制的桌面端下载器。
我们抛弃了繁琐的命令行和复杂的配置，提供了一个纯净、直观且极其稳定的图形化界面。不管你是单纯想快速存图的普通用户，还是有海量漫画管理需求的重度收集控，这款工具都能让你“下得爽，存得稳”。

---

## ✨ 我们到底能干嘛？ (核心功能)

这不是一个只会摆酷的空壳软件，它包含了你在下载漫画时需要的一切刚需功能：

*   **🔍 智能搜索与解析**
    - 支持直接在搜索框输入“关键词”寻找漫画。
    - 也支持直接粘贴 wnacg.ru 的“画廊网址”，程序会自动嗅探并拉取整本漫画信息。
*   **💾 极速下载与断点续传**
    - 支持多任务同时下载。即使突然断网、软件关闭、甚至电脑断电，下次打开时也能**接着原来的进度继续下**，一页都不会少。
*   **⏸️ 灵活的任务控制**
    - 你可以随时暂停、恢复、取消下载任务。
    - 支持 **Shift 连选**、**Ctrl+A 全选** 或是 **鼠标框选**，想要批量暂停或批量删除，只要选中后点一下鼠标。
    - **双击任务卡片** 即可快速切换暂停/恢复状态。
*   **📁 强迫症福音的本地管理**
    - 所有的漫画都会被整整齐齐地下载到软件目录下的 downloads/ 文件夹中。
    - 每个画廊会自动创建一个以其“标题”命名的独立文件夹。
    - 自动清理原图中的“透明底”并转换为体积更小的纯白底 JPG 格式，为您节省大量硬盘空间。
*   **⚙️ 丰富的配置选项**
    - 自定义并发数：网速慢？调低并发；千兆宽带？拉满火力！
    - 网络代理设置：支持直连，也支持手动指定本地代理节点。
    - 内置“一键更新”：点击更新按钮，程序会自动通过国内高速通道拉取最新版本并替换自己。

## 📦 极速上手 (Quick Start)

### 选项一：免环境直装版（适合绝大多数用户）
1. 前往右侧的 [Releases 页面](../../releases/latest)。
2. 下载 `WNACG-Downloader.exe`。
3. 把下载好的程序放到任意您喜欢的文件夹中（它会在该文件夹下自动生成 downloads 目录），双击运行即可畅快使用！（完全不需要安装 Python）

### 选项二：源码运行版（适合开发者）
推荐使用现代化的 Python 包管理器 [uv](https://github.com/astral-sh/uv) 来运行本项目。

``bash
# 1. 克隆代码仓库
git clone https://github.com/boo-yuan/wnacg-downloader.git
cd wnacg-downloader

# 2. 极速安装并运行
uv run src/main.py
`

## 💡 为什么它这么稳？(技术原理解析)

我们在底层做了一系列扎实的优化，不搞噱头，只为解决实际痛点：

1. **绝对不会被封 IP**：
   下载器内置了智能排队机制（也就是发牌器），它会让每一个图片的下载请求像真正的人类一样，带有随机的间隔停顿。这能彻底骗过网站的高级防爬虫系统，让你长时间下载也不会被拉黑。
2. **下载的图片绝对不会“只有半截”**：
   网络抖动经常会导致下载的图片出现下半截是灰色的。现在，下载器在每一张图片存入硬盘后，都会强制检测它的完整性。只要发现图片损坏，就会立刻扔掉重新下载，确保你的画廊 100% 完美。
3. **域名打不开也不怕**：
   如果你所在的地区突然打不开默认网址，程序会自动、静默地尝试去连接其他的备用节点（比如 .org, .net, .com），整个过程你甚至都察觉不到，下载依然丝滑。
4. **高速读写不卡界面**：
   采用了 SQLite 的 WAL（预写日志）模式。即使后台同时有几十个协程在疯狂读写数据库更新下载进度，你用鼠标滚动界面时，依然会感觉顺滑无比，不会有丝毫卡顿。
